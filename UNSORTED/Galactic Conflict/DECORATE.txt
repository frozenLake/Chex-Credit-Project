// Flem Mine (redefined to prevent friendly fire issues in ZDoom.exe) ----------

// It took me four years to figure out a work-around for this. I wanted to make
// it so that CQ3 could be usable with any version of ZDoom rather than be
// dependent on a special executable that was stuck in the past, and the coding
// for the Flem Mines broke their teamkilling ability. After a while, I happened
// to be reading about how A_SkullAttack was coded, and when I learned that the
// damage type is hard-coded as melee, the solution occurred to me: apply a zero
// damage factor for melee. Problem solved! The obvious side-effect is that
// you can't kill Flemoids with melee attacks, but with everything assuming a
// zorch damage type for this episode, that problem fixed itself. Chex Warrior
// can now actually do something Doomguy can't: melee-zorch Flemoids!

actor FlemMineV4 : FlemMine replaces FlemMine
{
	DamageType Flem
	DamageType Slime
	DamageFactor "Melee", 0.0
	States
	{
	Pain.Melee:
		Goto See
	}
}
